# List directories

	ls
	ls /home/shawn
	
## List hidden files

	ls -a
	
## Detailed list of files and directories

	ls -l

Can combine flags

	ls -la
	
## Show recursively

	ls -R
	
## Show reverse order

	ls -r
	
## Show by time (newest first)

	ls -t
